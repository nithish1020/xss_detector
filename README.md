# 🐺 XSS Detector  
Cross-Site Scripting Vulnerability Detection Tool  
Cyber Wolf Team · Mini Project · For Educational & Authorized Testing Only

## 📌 Overview
XSS Detector is a Python-based tool to detect Cross-Site Scripting (XSS) vulnerabilities.

## 📁 Project Structure
XSS-Detector/
├── xss_detector.py  
├── payloads.txt  
└── README.md  

## ✨ Features
- URL parameter scanning  
- HTML form scanning  
- HTTP header scanning  
- JSON & HTML reports  
- Payload-based detection  

## ⚙️ Requirements
pip install requests beautifulsoup4  

## 🚀 Usage
python xss_detector.py "http://example.com/search?q=test"
